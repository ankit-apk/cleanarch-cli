Quick Start

Get up and running with Clean Architecture CLI in minutes:

                        
# Install Clean Architecture CLI
go install github.com/yourusername/clean-architecture-cli@latest

# Generate a new project
clean-architecture-cli new myproject

# Navigate to your project
cd myproject

# Run your project
go run main.go

For more detailed instructions and configuration options, please refer to our GitHub repository.

About the Project

Clean Architecture CLI is an open-source project aimed at simplifying the process of creating Go applications with a clean and maintainable architecture. Our goal is to help developers quickly bootstrap projects that are scalable, testable, and adhere to best practices in software design.

We welcome contributions from the community! Whether you're fixing bugs, improving documentation, or proposing new features, your input is valuable. Check out our GitHub repository to get involved.
